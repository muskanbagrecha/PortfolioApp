BMTC Ticketing System

This is a ticketing system for booking seats in BMTC buses in Bangalore. It allows passengers to select the source and destination and book a seat in the buses running on that route. It also allows admins to add new routes and buses. Conductors can login and view the list of passengers who have booked a seat in the bus.


Technical details:

  • Used Flutter to develop the Mobile Application
  • Used Firebase to store the data

Palindrome Birthday

This app takes the user's birthday and checks whether it is a palindrome or not. A palindrome is a word, phrase, number, or other sequence of characters which reads the same backward as forward, such as madam. This app checks all the variations of a date format. In case the user's birthday is not palindrome, the app returns the nearest palindrome birthday and the number of days left till the nearest palindrome birthday.

Technical details:

  • HTML
  • CSS
  • VanillaJS
