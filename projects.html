Geek Finder

Geek Finder is a networking platform for students and professionals. It allows users to connect with people doing similar things as them. For instance, a person working on a project can collaborate with someone on Geek Finder based on the required tech stack. This is a project I worked on as a part of a team of 4.


Technical details:

  • Performed EDA, feature selection, feature engineering, text preprocessing, and encoding of categorical features
  • Applied weighted cosine similarity to find recommendations
  • Deployed the Flask Application on Heroku


BMTC Ticketing System

This is a ticketing system for booking seats in BMTC buses in Bangalore. It allows passengers to select the source and destination and book a seat in the buses running on that route. It also allows admins to add new routes and buses. Conductors can login and view the list of passengers who have booked a seat in the bus.


Technical details:

  • Used Flutter to develop the Mobile Application
  • Used Firebase to store the data
